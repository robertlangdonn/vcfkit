## What vcfkit is

vcfkit is a fast, open-source CLI for three operations every VCF pipeline needs —
normalize, liftover, and filter — rewritten for speed and correctness.
In one binary, measurably faster than bcftools on hot paths, and validated against it on
real 1000 Genomes data.

It doesn't replace bcftools. It does three operations well, with modern UX,
no htslib dependency, and an optional AI-powered natural-language layer that
translates plain English to filter expressions.

The same core runs in your browser via WebAssembly — paste a VCF in the demo
above and nothing uploads anywhere.
